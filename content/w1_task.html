w1_task

完成本節中的各項任務後, 表示你將可以登入:

  1. 學校 email 帳號郵箱
  2. Gmail 帳號郵箱
  3. Replit IDE 系統 (以 Gmail 帳號登入)
  4. Github 系統
  5. Onedrive 系統

登入學校 Email 郵箱:

學校 Webmail: https://mail.nfu.edu.tw (帳號為: 學號@nfu.edu.tw, 用來通知 stud.cycu.org 主機上的帳號密碼與學期成績)

申請或登入 Gmail:

假如尚未擁有 Gmail 帳號, 請自行申請一個 Gmail 帳號, 可用來登入 Replit 線上 IDE 系統.

申請 Github 帳號:

請利用"學號@nfu.edu.tw"在 Github 註冊一個帳號, 帳號名稱可以選擇學號, 或者用英文姓名的三個字母縮寫加上學號作為 Github 帳號. 帳號建立後必須進入 https://mail.nfu.edu.tw 回覆確認信件後, 才能開始使用 Github.

Github 帳號在計算機程式課程中的用途:

  1. 領取 Github Classroom 系統所指定的作業
  2. 建立個人網站: https://github_帳號.github.io
  3. 將計算機程式課程中所編寫的程式與相關資料存入 Github 帳號下的 Gist

在 Github 中, 將所建立的倉儲轉為網站, 可以在建立倉儲時, 以 https://github.com/mdecycu/cmsite 作為 template, 首先學習如何利用 Replit 維護網站內容, 隨後則可利用 stud 伺服器中的帳號進行 Github 倉儲與網站內容的維護, 最後也可以在近端可攜程式環境中維護 Gthub 倉儲與網站內容.

利用 Github 建立個人網站 (透過 Github Pages) 有以下目標:

  1. 可在網站中自我介紹 (https://www.betterup.com/blog/how-to-introduce-yourself)
  2. 可在網站中展示並介紹自己寫過的程式
  3. 可在網站中簡介自己修過的課程內容
  4. 可在網站中說明自己曾經參與過的專題
  5. 可在網站中介紹自己未來的生涯規劃

申請 onedrive 帳號:

  1. 登入學校郵箱 (用來收帳號建立確認信)
  2. 至 https://www.microsoft.com/zh-tw/education/products/office, 以 @nfu 帳號申請 OneDrive 帳號.

onedrive 帳號在計算機程式課程中的用途:

  1. 擁有線上 5TB 的儲存空間, 可用來儲存個人所建立的流程操作 mp4 影片, 並嵌入個人網頁或作業網站中
  2. 擁有線上 5TB 的儲存空間, 可用來儲存與課程相關的教育版授權軟體 (例如: Win10 22h2.4.iso, Win11 22h2.6.iso, kms.bat) 或可攜程式系統檔案與電子書
  3. 登入後可參與計算機程式課程 Teams (2023 計算機程式 MS Teams 代碼為: plmm43c)
  4. 登入後可下載計算機程式課程網站中的相關檔案

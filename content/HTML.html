Canvas: https://developer.mozilla.org/zh-TW/docs/Web/API/Canvas_API/Tutorial

上列圖形的程式碼:

<!-- add the canvas tag first -->
<p><canvas id="my_canvas" width="400" height="200"></canvas></p>
<script>// <![CDATA[
// Get the canvas element
        var canvas = document.getElementById("my_canvas");
        var ctx = canvas.getContext("2d");

        // Set line properties
        ctx.strokeStyle = "red";
        ctx.lineWidth = 5;
 
        // Draw the line
        ctx.beginPath();
        ctx.moveTo(50, 50); // Starting point coordinates (x, y)
        ctx.lineTo(350, 150); // Ending point coordinates (x, y)
        ctx.stroke();
// ]]></script>

HTML canvas and Javascript 練習:

  1. 請根據上列 html 與 Javascript 繪線程式的語法, 找出繪圖座標原點.
  2. 請利用與上列繪圖相同粗細的藍色線條, 從繪圖座標原點出發, 畫一個長度為 200 Pixel 的正方圖形.
  3. 請將上列正方形改為圓形.
  4. 請將圓形中央部分塗上黃色.
  5. 請利用黑色畫筆在圓形中央, 以 14 Pixel 大小的字元, 寫出"半徑 100 Pixel 的圓形"字串.

Brython_env

Browser Python - Python on Browser

在網頁中建立 Brython 程式環境: 

<!-- 導入 brython 程式庫 -->
<p>
<script src="./../cmsimde/static/brython.js"></script>
<script src="./../cmsimde/static/brython_stdlib.js"></script>
</p>
<!-- 啟動 Brython -->
<p>
<script>
window.onload=function(){
brython({debug:1, pythonpath:['./../cmsimde/static/','./../downloads/py/']});
}
</script>
<script type="text/python">
# write your Brython codes here
</script>

Brython modules or packages can put into downloads/py directory

A Python module is like a file containing special Python instructions that can be used by other Python files or programs. On the other hand, a script is a Python file that is meant to be directly run or executed.

Python Modules 範例:

例如: 將 point.py 放入 downloads/py 目錄中: 

#point.py
def point():
    return "可以傳回 point"

一旦導入上列引用 Brython 的頁面, 其中已經宣告 pythonpath 位於 static 與 downloads/py 目錄中, 因此 Brython 在頁面啟動時將導入當時位於這兩個目錄中的 .py 模組, 之後就可以透過 import 導入 point.py 作為模組: 

import point
print(point.point())

# 執行後的結果將會印出 downloads/py/point.py 中的 point() 函式執行內容, 也就是"可以傳回 point".

Python Packages 範例:

例如: 將 point2.py 放入 downloads/py/line/ 目錄中, 且在 line 目錄中加入一個空的 __init__.py 檔案.

且 point2.py 的內容與先前 point.py 的內容相同. 一旦上列導入 Brython 且宣告 pythonpath 位於 downloads/py 之後, 之後頁面中的 Brython 程式就可以採 import 的方式導入 line package, 並且使用 package 中的 point2.py modules. 

# 引用 line package 的範例
from line import point2
print(point2.point())

# 執行結果會印出"可以傳回 point"

Brython 是一個讓您可以使用 Python 程式語言在瀏覽器中執行的工具。以下是一個簡單的說明，可用於向初學者解釋 Brython：

Brython 是 "Browser Python" 的縮寫，它是一個專為瀏覽器設計的 Python 執行環境。Python 是一個簡潔易讀且強大的程式語言，而 Brython 讓您能夠直接在瀏覽器中運行 Python 程式碼，而不需要任何額外的安裝或設定。

使用 Brython，您可以在網頁上編寫 Python 程式碼，並在瀏覽器中執行。這對於學習 Python 或在網頁上進行簡單的動態互動非常有用。您可以編寫各種功能，例如計算器、遊戲、表單驗證等等，並在瀏覽器中立即查看結果。

Brython 的優勢之一是它提供了與標準 Python 類似的語法和功能。這意味著您可以使用您已經熟悉的 Python 語法來編寫瀏覽器腳本，而不需要學習其他語言或概念。它支援大部分的 Python 3 語法和標準庫，這讓您可以使用許多內建函式、資料結構和模組。

使用 Brython，您可以將 Python 程式碼直接放在網頁的 "<script type='text/python'>" 標籤中，或者使用外部腳本檔案引入。Brython 會解析並執行您的 Python 程式碼，並將結果顯示在網頁上。

總結來說，Brython 是一個讓您在瀏覽器中運行 Python 程式碼的工具。它使得在網頁上使用 Python 變得容易，並提供了與標準 Python 相似的語法和功能。如果您想要在網頁上進行簡單的動態互動或學習 Python，Brython 是一個很好的選擇。
